# CRUD - TODO  

A crud operations todo made with React.js Framework.


## Features

- Easy to integrate with Backend
- Fully Responsive



## Run Locally


Go to the project directory

```bash
  cd todo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* [React](https://reactjs.org/)
